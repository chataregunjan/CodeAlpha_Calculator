# CodeAlpha Calculator

A simple and responsive calculator developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features
- Addition, Subtraction, Multiplication, and Division
- Clear (C) button
- Real-time calculations
- Responsive design for mobile and desktop
- Clean and modern user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Objective
To create a functional calculator that performs basic arithmetic operations while providing an intuitive and responsive user experience.
